# TradingAssistantX 要件定義書

## 🎯 システム概要

**独自見解による高付加価値投資情報配信システム**

X（旧Twitter）上で金融市場への独自解釈・業界内幕・データドリブン予測を配信し、投資初心者〜上級者・金融関係者への差別化された専門情報提供を実現するシステム。

### 核心価値
- **独自性担保**: リアルタイム金融情報への独自解釈・業界内幕の提供
- **予測精度**: データドリブンな根拠ある市場予測・リスク警告
- **反常識視点**: 一般論への疑問提起・逆張り思考による差別化
- **検証透明性**: 予測結果の公開検証・失敗からの学習
- **24時間運用**: スケジュール実行による継続的価値提供
- **学習進化**: 実行結果から翌日戦略を自動最適化

## 📋 基本機能要件

### 実行モード
- **開発実行**: `pnpm dev` - 単一実行（手動テスト用）
- **本番実行**: `pnpm start` - スケジュール実行（自動運用）

### コアアクション
- **投稿 (post)**: リアルタイム独自分析・予測・警告・業界内幕の投稿
- **リツイート (retweet)**: 高品質コンテンツの拡散
- **引用リツイート (quote_tweet)**: 金融ニュースへの独自解釈・反常識視点の提供
- **いいね (like)**: 関係構築・品質評価
- **フォロー (follow)**: 投資・金融関連アカウントとの専門ネットワーク構築
- **分析 (analyze)**: 深夜分析・翌日戦略生成（23:55限定）
- **待機 (wait)**: 最適タイミング調整

## 👥 対象ユーザー層

### **投資初心者～上級者・金融関係者**

#### 初心者向け価値提供
- **業界内幕・実情暴露**: 金融業界の裏側・投資詐欺の最新手口
- **実践的リスク警告**: バブル兆候の早期発見・危険銘柄の具体的指摘
- **感情論への科学的反論**: 行動経済学による投資心理分析

#### 中級者向け価値提供  
- **独自分析手法**: オリジナルフレームワーク・新しい投資判断基準
- **予測手法**: データドリブンな根拠ある市場分析
- **逆張り視点**: 一般論への疑問提起・反常識的思考

#### 上級者・金融関係者向け価値提供
- **高度な市場分析**: 中央銀行決定の裏側分析・経済指標の真の意味解説
- **機関投資家動向**: ファンドマネージャーの本音・機関投資家の実情
- **独自市場解釈**: 市場予想との乖離要因分析・独自の市場解釈モデル

## 📊 成功指標

### **従来指標**
- **エンゲージメント率**: いいね・リツイート・コメント数
- **フォロワー増加率**: 月次フォロワー増加数・増加率

### **差別化指標**
- **独自性評価**: 他投資アカウントとの差別化度・オリジナル分析の頻度
- **予測精度**: 市場予測の的中率・予測根拠の透明性・外れた予測の検証公開
- **専門性認知**: 金融関係者からの評価・業界専門メディアからの引用数
- **リアルタイム性**: 重要経済ニュースへの反応速度・独自見解提供の迅速性
- **影響力指標**: 投稿内容の他メディア・アカウントへの引用・参照頻度

## 🏗️ システムアーキテクチャ

### ワークフロー（全アクション共通）
```
1. データ収集 → 2. アクション実行 → 3. 結果保存
```

**アクション実行詳細**:
- **通常アクション（post, retweet, like等）**: 通常の3ステップで完了
- **analyzeアクション（23:55のみ）**: 深夜分析を実行
  - **詳細仕様**: [docs/deep-night-analysis.md](docs/deep-night-analysis.md)

### 核心コンポーネント
- **Claude Code SDK**: AI判断・コンテンツ生成
- **KaitoTwitterAPI**: X（Twitter）連携・投稿管理
- **DataManager**: データ統合管理（current/history 2層構造）
- **TimeScheduler**: 時刻ベース自動実行

## 🤖 Claude Code SDK統合

**詳細仕様**: [docs/claude.md](docs/claude.md)

### エンドポイント別設計
- `generateContent`: 独自解釈・予測・逆張り視点・業界内幕コンテンツ生成
- `selectOptimalTweet`: AI最適選択（独自性・専門性・予測価値・エンゲージメント総合判断）
- `generateQuoteComment`: 反常識視点・独自分析による価値追加コメント生成
- `analyzePerformance`: 予測精度検証・実行結果分析・戦略最適化

### 深夜分析システム
- **実行時刻**: 毎日23:55
- **詳細仕様**: [docs/deep-night-analysis.md](docs/deep-night-analysis.md)

## 🌐 KaitoTwitterAPI連携

**詳細仕様**: [docs/kaito-api.md](docs/kaito-api.md)

### 2層認証アーキテクチャ
- **読み取り専用**: APIキー認証（検索・情報取得）

### API制限対応
- **レート制限**: 200 QPS
- **プロキシ管理**: 自動ローテーション
- **エラーハンドリング**: 自動リトライ・継続実行

## 🕒 実行仕様

**詳細フロー**: [docs/workflow.md](docs/workflow.md)

### スケジュール例
- **07:00**: 朝の市場予測・独自見解投稿
- **12:00**: リアルタイム市場分析・逆張り視点投稿
- **20:00**: 引用RT（金融ニュース独自解釈・業界内幕暴露）
- **23:55**: **深夜分析**（予測検証・翌日戦略生成）

### 実行制御
- **時刻精度**: 1分間隔チェック
- **エラー継続**: 個別失敗でもスケジュール継続
- **自動学習**: 実行結果から戦略最適化
